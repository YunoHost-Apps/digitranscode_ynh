## Aurreikuspena

Digitranscode is a simple application to convert audio and video files thanks to FFMPEG.wasm - https://github.com/ffmpegwasm/ffmpeg.wasm (MIT License).


**Paketatutako bertsioa:** 0.2.0~ynh3
